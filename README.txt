This code represents an incomplete implementation of a C-based HTTP server for educational purposes. It is capable of handling basic GET requests and file-based routing. Feel free to contribute to this project by adding more features or fixing bugs.

Usage
=====

To run the server, simply run the following commands to compile the server:
1. For make:

```
make
2. For gcc (using build.sh):
```

chmod +x build.sh && ./build.sh
```
    
Then run the server using the following command:

```
./http_server <address> <port>
```
